# OpenAI Docs Eval Case

## case-id: latest-model-choice

目标能力：

当用户问“现在该用哪个 OpenAI 模型”时，能基于官方文档给出带日期的答案，而不是凭记忆回答。

输入：

```text
我现在做一个中文客服助手，想接 OpenAI API。
请告诉我现在该怎么选模型，为什么，以及需要注意什么。
```

期望行为：

- 明确提示需要查官方文档并给出最新来源。
- 按场景解释模型选择、限制和迁移注意点。
- 说明“截至查询时”的时间点。

失败判定：

- 直接输出过时模型名却不说明来源。
- 没有任何官方链接。
- 把猜测当成最新事实。
